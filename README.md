# ✦ ホロスコープ作成アプリ

生年月日・生まれた時間・出生地から、本格的な**西洋占星術シート**（元素・3区分・陰陽のバランス表）を作成し、一流占星術師スタイルの鑑定文を添えて表示する無料Webアプリです。

## 特長

- **西洋占星術シート**：10天体を12サインに配置し、四元素（火・土・風・水）／3区分（活動・固定・柔軟）／陰陽のバランスを集計（添付Excel「西洋占星術シート」準拠）
- **鑑定文**：太陽・月・水星・金星・火星の配置、元素バランス、陰陽の偏りから流れのある日本語の解説を生成
- **生まれた時間が不明な人向け**：チェックボックスで「正午に生まれた」と仮定して計算
- **参考占い**：四柱推命・動物占い・六星占術・五星三心占いも同時表示

## 技術

- 天体位置の計算：[astronomy-engine](https://github.com/cosinekitty/astronomy)（MIT）でトロピカル方式（その時点の春分点基準）の黄経を精密計算
- 四柱推命の節入り：太陽黄経で厳密判定（立春＝太陽黄経315°）
- 動物占い／六星占術／五星三心：生年月日から導く干支ナンバー（1〜60）方式
- ビルド不要の静的サイト（HTML/CSS/JS）

## 注意

星の配置は「設計図」であり、未来を断定するものではありません。東洋系の占いは暦からの参考算出で、各占術の正式鑑定とは細部が異なる場合があります。エンタメとしてお楽しみください。
